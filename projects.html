A food ordering web application
The Food Ordering Website is a dynamic and user-friendly web application that allows users to explore a variety of delicious cuisines and place orders seamlessly. Built using the MERN stack, which comprises MongoDB, Express.js, React, and Node.js, this website aims to provide a delightful food ordering experience to its users.The Food Ordering Website is designed to be fully responsive, ensuring a seamless experience across various devices, including desktops, tablets, and mobile phones.
Front-End: HTML, CSS, Bootstrap, JavaScript, React
Back-End: Node.js, Express.js
Database: MongoDB
